# Portfolio

## Typography (Saans)

This site is configured to use `Saans` for all typography.

To ensure the font renders the same on every device, add these files:

- `assets/fonts/Saans-Regular.woff2` (400)
- `assets/fonts/Saans-Medium.woff2` (500)
- `assets/fonts/Saans-SemiBold.woff2` (600)

If you don’t add them, the site will fall back to system fonts.
